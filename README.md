# Fuel Station Data Management System

This project models a multi-gas-station fuel pricing system using XML, JSON, and a web-based interface.

---

## Project Overview

The system is inspired by real-world fuel price platforms (e.g., fuelGR) and demonstrates structured data modeling, transformation, and dynamic web data loading.

---

## Features

* XML structured dataset for multiple gas stations
* Nested fuel information per station
* JSON conversion of XML data
* DTD validation support
* Dynamic web interface (AJAX-based)
* XML and JSON data rendering in browser

---

## Data Structure

### Gas Station Includes:

* Company information
* Location & coordinates
* Distance to user
* Operation hours (24h support)
* Contact phone(s)
* Fuel types and prices

### Fuel Includes:

* Fuel name
* Fuel code
* Price per liter
* Timestamp of last update

---

## Technologies Used

* XML (data modeling)
* DTD (validation)
* JSON (data transformation)
* HTML / CSS / JavaScript
* AJAX (dynamic loading)

---

## How to Run

### Option 1 (Recommended)

Use GitHub Pages or Live Server to enable AJAX functionality.

### Option 2

Open `index.html` directly (limited functionality due to browser security restrictions).

---

## Files

* fuel.xml → structured XML dataset
* fuel.json → converted JSON version
* index.html → web interface
* script.js → AJAX logic
